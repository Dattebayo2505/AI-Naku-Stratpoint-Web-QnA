## Summary

[BoxCrush,](https://www.boxcrush.com/) a US-based web design, development, marketing, and server management company with more than 23 years of experience, continuously enhances its digital infrastructure to support its growing client base. Stratpoint optimized their AWS Cloud infrastructure, eliminating costs to maintain uptime, reducing CPU utilization to 10%, and achieving annual cost savings of more than $20,000. This allows BoxCrush to focus on delivering exceptional digital experiences to their clients.

## Challenges

BoxCrush leveraged Amazon Web Services (AWS) to host their customers' websites. While AWS offered scalability and performance, the rapid growth of the BoxCrush portfolio led to some complexities:

* High CPU utilization resulting in higher AWS costs
* To ensure there was no website downtime, BoxCrush increased spending to maintain a high level of performance

## Solutions

Stratpoint deployed a team of certified AWS architects and engineers to provide Managed Cloud Services, implementing crucial changes to enhance performance and security.

The team implemented changes to mitigate performance and security issues on the AWS platform, such as:

* Resizing instances according to the changing needs of the application
* Migrating old AWS products and resources to current versions
* Re-architecture — separation of apps and database instances
* Operating systems (OS) upgrades
* Enabling AWS services updates and upgrades
* Upgrading storage volume
* Additional security through web application firewall
* Proactive alarms per AWS resource
* Status alarms for URLs/websites
* Dashboard for resource monitoring
* Tagging strategy to enable BoxCrush to see the AWS cost per customer across multiple instances and AWS services
* Removal of unnecessary AWS resources
