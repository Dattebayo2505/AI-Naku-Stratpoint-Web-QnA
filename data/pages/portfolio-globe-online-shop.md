E-Commerce Web Application

### Globe increases online channel revenues through Agile Development

![](https://stratpoint.com/wp-content/uploads/portfolio-globe-store-1-scaled.webp)

Background

The Globe Online Shop is an **e-commerce website** where customers can purchase postpaid, prepaid, and broadband subscriptions, as well as mobile devices, gadgets, and electronic accessories.

Solution

Stratpoint deployed a team to develop, test, and deploy all the features and capabilities that make up the Globe Online Shop — <https://shop.globe.com.ph/>. The implementation was based on Agile methodology, so they were able to launch quickly and release new features and updates frequently. Stratpoint handled the project end-to-end, including post-live issue management.

Outcome

Aside from boosting the brand’s presence, the Globe Online Shop significantly contributes to its online channel revenue, with customers embracing the convenience of shopping remotely. Customers may expect the site’s interface and resulting experience to get better with every update.

Technologies used

* PHP, Javascript
* CSS (LESS) / HTML
* Nginx, PHP-FPM
* OpenCart
* AWS Services: EC2: RDS, SES, S3, Cloudfront, DynamoDb
